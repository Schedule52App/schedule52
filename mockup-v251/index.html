All scenarios use the real Wilbanks logo. The right-side user chip is illustrative only. Resize this window to ~1280px or 1440px to see how it'd render on your normal laptop.

Implementation notes:
• The band is a new component (<TopBrandBand />) mounted above the existing <Sidebar /> in the app shell.
• Existing logo_scale_dashboard setting retargets to band height (sm=56 / md=72 / lg=96) — no schema change.
• Render mode still ships: icon shows logo + text; lockup shows logo only (hides name/tagline because the logo includes them); none shows text only.
• logo_scale_invoice stays as-is — it only affects the invoice/estimate email templates.
• Mobile header keeps its current full-width treatment (it already does the right thing).
• No backend changes. Only Sidebar.tsx + App.tsx (or wherever the shell lives) get edits.
